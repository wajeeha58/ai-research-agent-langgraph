# AI Research Assistant using LangGraph

This project demonstrates an AI research assistant built using LangGraph and LangChain.

The system:
- Analyzes user questions
- Searches the web using DuckDuckGo
- Uses an LLM to generate answers

## Tech Stack

Python  
LangGraph  
LangChain  
Groq Llama Model  
DuckDuckGo Search  

## Example

Input:
Explain Agentic AI

Output:
Agentic AI refers to artificial intelligence systems capable of autonomous decision-making and goal-oriented actions.
